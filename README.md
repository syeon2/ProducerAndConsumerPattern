## Producer And Consumer Pattern

### 구현 목적
Producer와 Consumer 패턴을 구현하면서 자연스럽게 접하게 되는 Java Thread의 특징과
mutex, monitor, synchronized 등의 동기화 기술, Thread의 전체적인 흐름 그리고 동기화를
위해 부가적으로 알아야할 기술들(volatile, AtomicXx 등)을 파악한다.
